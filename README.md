# 📝 To-Do List App

Um aplicativo de lista de tarefas moderno e interativo, desenvolvido com JavaScript puro, com foco em organização, experiência do usuário e estrutura de código limpa.

---

## 🚀 Funcionalidades

- ✅ Adicionar tarefas
- 🗑️ Remover tarefas
- ✔️ Marcar como concluída
- 🗂️ Organização por categorias:
  - Trabalho
  - Casa
- 🔎 Filtro por categoria (sidebar)
- 🎨 Interface moderna e responsiva
- ⚡ Interações suaves (animações e hover)

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  

---

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar:

- Manipulação de DOM
- Estruturação de aplicações front-end
- Organização de código
- Criação de interfaces interativas
- Pensamento de produto (UX/UI)
